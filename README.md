# React App

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build:prod
```

To create a development build:

```sh
npm run build:dev
```

To run a localhost in development mode:

```sh
npm run start
```

To run a esLint:

```sh
npm run lint
```

To run a Prettier:

```sh
npm run format
```
